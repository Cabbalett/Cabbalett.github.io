# Function & Console IO
## Function
- 어떤 일을 수행하는 코드의 덩어리
- 반복적인 수행을 1회만 작성 후 호출할 수 있다.
- 코드를 논리적인 단위로 분리할 수 있다.

## Console IO
### 콘솔창 입력 - input
- input() 함수는 콘솔창에서 문자열을 입력 받는 함수
### 콘솔창 출력 - print
- 콤마를 통해 print를 연결할 수 있다.
{% highlight ruby %}
>>> print("Hello World!", "Hello Again!!!")
Hello World! Hello Again!!!
{% endhighlight %}
### Print Formatting
- 프린트 문은 기본적인 출력 외에 출력 형식을 지정할 수 있다.
{% highlight ruby %}
print(1,2,3)
print("a"+" "+"b"+" "+"c")
print("%d %d %d" % (1,2,3))
print("{} {} {}".format("a","b","c"))
print(f"value is {value})
{% endhighlight %}

> %-format
> type | 설명
> ---- | ----
> %s | String
> %c | Character
> %d | Integer
> %f | Floating-point
> %o | 8진수
> %x | 16진수
> %% | Literal %(문자 % 자체)
