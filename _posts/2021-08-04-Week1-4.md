# Python Data Structure
### Stack
- 나중에 넣은 데이터를 먼저 반환하도록 설계된 메모리 구조
- Last In First Out(LIFO)
- Data의 입력은 Push, 출력을 Pop이라고 함
- list로 구현 가능

{% highlight ruby %}
>>> a=[1,2,3,4,5]
>>> a.append(10)
>>> a.append(20)
>>> a.pop()
20
>>> a.pop()
10
{% endhighlight %}

### Queue
- 먼저 넣은 데이터를 먼저 반환하도록 설꼐된 메모리 구조
- First In First Out(FIFO)
- Stack과 반대되는 개념
- list로 구현 가능

{% highlight ruby %}
>>> a=[1,2,3,4,5]
>>> a.append(10)
>>> a.append(20)
>>> a.pop(0)
1
>>> a.pop(0)
2
{% endhighlight %}

### Tuple
- 값의 변경이 불가능한 리스트
- 선언 시 "[]"가 아닌 "()"을 사용
- 리스트의 연산, 인덱싱, 슬라이싱 등을 동일하게 사용

{% highlight ruby %}
>>> t =(1,2,3)
>>> print(t +t ,t \*2)
(1,2,3,1,2,3)(1,2,3,1,2,3)
>>> len(t)
3
>>> t[1]=5

{% endhighlight %}

### Set
- 값을 순서없이 저장, 중복 불허 하는 자료형
- 객체 선언을 이용하여 객체 생성
  
{% highlight ruby %}
>>>s =set([1,2,3,1,2,3])
>>>s
{1,2,3}
>>>s.add(1)
>>>s
{1,2,3}
>>>s.remove(1)
>>>s
{2,3}
>>>s.update([1,4,5,6,7])
>>>s
{1,2,3,4,5,6,7}
>>>s.discard(3)
>>>s
{1,2,4,5,6,7}
>>>s.clear()
{% endhighlight %}
