# Pythonic Code

- 파이썬 스타일의 코딩 기법
- 파이썬 특유의 문법을 활용하여 효율적으로 코드를 표현
- 고급 코드를 작성할수록 더 많이 필요해짐

예시: 여러 단어들을 하나로 붙일 때

```json
>>> colors = ['red','blue','green','yellow']
>>> result = ''
>>> for s in colors:
      result +=s
```

```json
>>> colors = ['red','blue','green','yellow']
>>> result = ''.join(colors)
>>> result
'redbluegreenyellow'
```
